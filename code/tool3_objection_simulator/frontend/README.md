# Tool 3 Frontend

This is the React/Vite client for Tool 3, Enterprise Objection Simulator.

For complete setup, backend instructions, environment variables, demo flow, and troubleshooting, see [the tool README](../README.md).

Quick frontend commands:

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 5175
npm run build
```

The client expects the Tool 3 backend at:

```text
http://localhost:8002
```
