# Tool 1 Frontend

This is the React/Vite client for Tool 1, Architecture Diligence & Wrapper Detection.

For complete setup, backend instructions, environment variables, demo flow, and troubleshooting, see [the tool README](../README.md).

Quick frontend commands:

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 5173
npm run build
```

The client expects the Tool 1 backend at:

```text
http://localhost:8003
```
